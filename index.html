Our Friend Leo

Product
Website
2012

Leo is a guitar with a personality and a voice to express it. The Duncan/Channon Makers gave him sensors, a microcontroller, and a Twitter account. With these, he can live-tweet his guitar experiences any time something happens, like when someone plays him.
—
Visit ourfriendleo.com.
Collaborators: Hannah Puckett, Timothy Trieu, and Robin Yang.

Splitter

Splitter allows you to split group expenses, without the hassle. Built using Ratchet and deployed as a web app using Heroku.

Collistive

Website
2012

Collistive crowdsources the most popular items in a list. Users can upvote or downvote items, add new items to an existing list, or create new lists. Launched in alpha in the spring of 2012 as a prototype for a web-based product development course.
—
Visit collistive.com.
Collaborators: Gene Pang, Genevieve Wang, Edmund Ye.

Berkeley Innovation

Identity
Website
2012

Identity and website for Berkeley Innovation, an undergraduate human-centered design group. The identity is set in Lucida Grande, with orange accent. The website embeds the latest posts, photos, project updates, and resources from the group's social media accounts (including Twitter, Flickr, Behance, and Bitly), enabling the site to remain dynamic with little website maintenance.
—
Visit Berkeley Innovation's website.
